SARAS ERP v2 — Masters

Pehle registers banao, phir kaam shuru

(First set up your registers, then start working)

Masters = Aapke Permanent Registers

Jaise aapke paas ek Customer Bahi hai, ek Supplier Bahi hai — waise hi ERP mein Masters hain.

Ek baar set karo, phir har jagah automatically use hote hain.

Translation: Masters are like your permanent registers. Just like you have a Customer register and Supplier register, ERP has Masters. Set them up once, they auto-fill everywhere.

📝 Master Register → ✅ Orders, 📊 Calculator, 🧾 Invoice, 📦 Dispatch, 💳 Payments
Set once, use everywhere!
👥
Group 1: Log Kaun Hai?
(Who are the people?)
Jinse deal karte ho — customers, suppliers, brokers, staff
👤 Customer Register
Jisko maal bechte ho (Who you sell to)
Fields: Firm name, contact person, phone, WhatsApp, address, GSTIN, credit limit, payment terms, broker info
Orders Invoice Dispatch Payments
📦 Supplier Register
Jisse maal lete ho (Who you buy from)
Fields: Firm name, phone, address, GSTIN, materials supplied, rates, quality rating, bank details
Purchase Stock Calculator
🤝 Broker Register
Dalaal/Agent
Fields: Name, phone, commission percentage
Orders
👔 Staff Register
Apne log (Your team)
Fields: Name, phone, role, access permissions, department
Login Permissions Everywhere
⚙️ Operator/Worker Register
Kaarigars (Machine operators)
Fields: Name, phone, daily wage, machines they operate, processes they know
Calculator Production
🧵
Group 2: Kya Banate Ho?
(What do you make?)
Aapke products, machines, materials — sab yahan set karo
🔧 Machine Types
Aapki machines
Example: 8 Spindle Round, 13 Spindle Flat, 48 Spindle Rope, etc.
Fields: Name, custom number, spindle count, type, speed, hourly cost, photo
Calculator Production Job Cards
📌 Product Types
Kya items banate ho
Example: Hollow Cord, Round Cord + Filler, Flat Elastic, Shoelace, etc.
Fields: Name, type, needs filler or not, suitable machines, HSN code
Calculator Orders Invoice
🌀 Chaal Types
Braiding pattern
Example: Seedhi Chaal, Jodi Chaal, Guthai, etc.
Fields: Name, speed factor, pattern code
Calculator
🎨 Yarn / Material Register
Dhaga/material ki list
Example: Cotton 20s, Polyester DTY 600D, Red Filler, Spun Viscose 30s, etc.
Fields: Name, type, count/denier, rate ₹/kg, supplier, stock quantity, color
Calculator Purchase Stock Orders
⚙️
Group 3: Kaam Kaise Hota Hai?
(How does work happen?)
Manufacturing ke steps — cone winding se leke cutting tak
📋 Process Types
Manufacturing ke steps
Example: 1. Cone Winding → 2. Bobbin Winding → 3. Braiding → 4. Tipping/Cutting
Fields: Name, sequence order, required machine, time estimate
Calculator Production Job Cards
✅ Quality Parameters
Quality check kya dekhna hai
Example: Weight per meter, width, strength, color matching
Fields: Parameter name, min/max value, mandatory or optional
Quality Check Production

Invoice Payments
🏷️ Charge Types
Extra charges

Fields: Name, fixed amount or percentage

📏 Units
Naap ke units
Example: meter, kg, gram, piece, dozen, set, box
Fields: Unit name, conversion factor
Everywhere

🔗 Kahan Kahan Use Hota Hai?

Har module ko kaun-kaun se masters chahiye?

(Which masters does each module need?)

📝 Orders / Sales
Uses: Customer, Broker, Product, Yarn, Machine, Chaal, Payment Terms, Charge Types, Units
🧮 Calculator
Uses: Machine, Product, Yarn, Process, Chaal, Operator, Units
📊 Production / Job Cards
Uses: Machine, Product, Process, Operator, Yarn, Quality Parameters
📦 Stock / Inventory
Uses: Yarn, Product, Warehouse, Color, Units, Packaging
🚚 Dispatch / Shipping
Uses: Customer, Product, Warehouse, Packaging, Transport, Units
🧾 Invoice / Billing
Uses: Customer, Product, HSN Codes, Banks, Charge Types, Payment Terms, Units
💰 Payments
Uses: Customer, Supplier, Banks, Payment Terms
🔐 Access Control
Uses: Staff, Roles & Permissions
